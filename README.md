# DRCore 2
Core Classes for my game developing, inspired by Tribase Engine from David Scherfgen
This Core Classes haven't any dependencies at other libs

This library contain basics like: 
- Vector classes
- Matrix
- Plane
- Logging
- Game State Manager
- Cubic Spline
- Bezier Curve
- Damping Spring
- Fast ini-File Parser

build with 
```bash
mkdir build
cd build
cmake .. 
make 
```
